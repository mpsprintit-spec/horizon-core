# Examples Module

## Overview

The examples module provides representative Indonesian sentences that demonstrate dictionary entries, grammar rules, syntax patterns, semantic interpretation, pragmatics, conversation, expressions, idioms, and abbreviations. It is not a corpus.

## Representative examples

| Example | English gloss | Demonstrates | Cross-reference |
| --- | --- | --- | --- |
| `Saya sudah makan.` | I have already eaten. | completed aspect with `sudah` | grammar |
| `Kami akan berangkat besok.` | We will leave tomorrow. | future marker `akan` | dictionary `akan`; grammar |
| `Ada dokter di klinik.` | There is a doctor at the clinic. | existential clause | dictionary `ada`; syntax |
| `Pintu dibuka oleh petugas.` | The door is opened by the officer. | passive with `di-` | grammar; syntax |
| `Dokumen itu sudah saya baca.` | I have read that document. | topic-fronted patient | syntax |
| `Bisa bantu saya?` | Can you help me? | ability form used as request | semantics; pragmatics; conversation |
| `Maaf, saya kurang setuju.` | Sorry, I somewhat disagree. | softened disagreement | pragmatics; conversation |
| `Silakan masuk.` | Please come in. | polite invitation | expressions; pragmatics |
| `Hati-hati, lantainya licin.` | Be careful, the floor is slippery. | warning and safety intent | expressions; pragmatics |
| `Dia membawa buah tangan dari Bandung.` | They brought a souvenir from Bandung. | idiom `buah tangan` | idioms |
| `Mohon isi nomor KTP.` | Please fill in the identity-card number. | administrative abbreviation | abbreviations |
| `Informasi itu benar, bukan salah.` | That information is correct, not wrong. | antonymy | semantics |

## Annotation standard

Future examples should include:

- Indonesian sentence.
- English gloss.
- Primary module demonstrated.
- Relevant cross-references.
- Register or context when the sentence is not neutral.

Examples should be short, natural, and reusable. Avoid long dialogues, artificial bulk generation, and duplicate examples already owned by another module.
